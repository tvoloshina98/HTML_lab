Лабораторная работа N 2.6
Изменение свойств таблицы

Задание

Создайте форму в виде таблицы, состоящей из:

  • текстового поля для ввода значения border таблицы;
  • текстового поля для ввода значения cellSpacing таблицы;
  • текстового поля для ввода значения cellPadding таблицы;
  • кнопки "Изменить", по нажатию которой производится изменение свойств таблицы.

Пояснение.
Использовать следующий способ доступа к элементам форм. При задании тегов можно применять параметр ID.
Если в теге TABLE задан параметр ID="имя", то модификация размера рамки тавлицы произойдет
при изменении свойства document.all("имя").border.
